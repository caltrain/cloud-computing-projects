1. Copy the src into the twister samples folder
  
2. Compile the Samples using ant build file
  
3. Copy the jar file generated in the dist/lib folder into twister apps folder
  
4. Run twister power make up script
  
5. Start twister using ./startTwister.sh script
  
6. Generate the data using kmeans data gen script
  
7. Create partition file using the given sample input file: "apriori_sample_input.txt"
  
8. Run the program using run_apriori.sh script
  
9. output will be generated in apriori_output.txt file as on the console
